# DineFlow

Restaurant Management System Using Django Framework

## About the Project

DineFlow is a web-based Restaurant Management System developed using Django Framework. The application helps restaurants manage customers, menu items, food orders, billing, and administrative operations through a centralized platform.

## Features

* User Registration and Login
* Customer Profile Management
* Menu Management
* Food Ordering System
* Order Tracking
* Billing Management
* Admin Dashboard
* Responsive User Interface

## Technologies Used

* Python
* Django
* HTML
* CSS
* Bootstrap
* JavaScript
* SQLite
